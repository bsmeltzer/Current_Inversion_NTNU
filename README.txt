This repository analyzes sea surface images of the wave field to infer the underlying currents.

The code extracts wavenumber-dependent Doppler shift velocities and finds the vertical current profile.

All code is in MATLAB.

External libraries: CopterCurrents, available:
https://github.com/RubenCarrascoAlvarez/CopterCurrents


See 'EXAMPLE_PROCESS_CUBE' to get started.
